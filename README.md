# ILS Product — Facilitator Email Templates

Email templates used by Stratton Index for the ILS facilitator certification process.

---

## Files

### `Facilitator Welcome Email.html`

Sent to new facilitators when they sign up for ILS certification. Opens the onboarding sequence.

**To use:**
- Open the file in any browser to preview it as it will appear in an email client.
- Copy the `<table>` block (starting at the comment `THE EMAIL ITSELF`) and paste it into your email tool (Gmail, Outlook, etc.).
- Replace the mail-merge placeholders before sending:
  - `{{FirstName}}` — recipient's first name
  - `{{LastName}}` — recipient's last name
  - `{{Email}}` — recipient's email address
- The group code (`DJP26`), promo code (`open`), and resource password (`ilsresources`) are fixed and already in the template.
- The "From / To / Cc / Attached" header above the email card is a preview aid only — do not copy it.

**Attachments to include when sending:**
- Stratton Index Facilitator Guide (PDF)
- ILS Workshop Deck ×2 (PPTX)
- ILS Debrief Deck (PPTX)
- Recipient's ILS Report (PDF)
- ILS Licensing Agreement (PDF)
