# Prompt: Add a New Design to the Resources Page

Use this prompt with Claude whenever you have a new design to add.

---

I have a new design to add to the resources page. Here's the link: [paste design URL]

Please fetch it, read it, and add it to the `stratton-index-ils-product` repo. Based on the content of the design, suggest an appropriate card title, card description (one sentence, describing when to use it), and tags. If it fits into an existing section on the index page, add it there — if not, suggest a new section name. Show me your suggestions before making any changes so I can confirm or adjust.
